To run the code locally, run:

``` 
Git clone git@github.com:Sanderchriss/CodeClubWebsiteWorkshop.git
```
Or download the zip-file on the green button above.

After you have cloned it, run `npm install`, then `npm run dev` to start the server.

This stack runs on Vite/Typescript, React and Material-UI for easy design components.
